In order to make things easier for the user to manage dice rolls, card draws, and feel overall more snappy I've created a Chrome Extension that will parse text and display it, allowing for snappy dice rolls and deck draws allowing you to make draws, copy it, and relay it to the bot

It tracks variables across rolls so that features link together (hex roll effects hex draw hand size) and features a copy button to relay them to the bot

Download it here!

It may be updated in the future with more features (music, character sheet, inventory) but for now it's just a simple extension to make it more convenient to use.

You can also find more features (such as constantly updating maps, journals, and otherwise that might be useful to you here, I did not bundle them with this extension because I was having issues with integrating them and couldn't figure it out.
