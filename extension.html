In order to make things easier for the user to manage dice rolls, card draws, and feel overall more snappy I've created a Chrome Extension that will parse text and display it, allowing for snappy dice rolls and deck draws allowing you to make draws, copy it, and relay it to the bot

How to use

For Dice & Card mechanics, make sure you utilize the Manual versions of all modules in the Home section of the website. You can then click the button in the top left, it will parse the message for any uses of dice rolls or hexes, ensure that it follows the proper format (Read "Deadlands: Custodial Edition Intro" on the sidebar for more details)

For the inventory module, use the below system prompt in place of the default inventory module, and ensure that you update your chat memory constantly. You can paste your chat memory template, companion template, and any other details you keep in your Chat Memory section into the 'Copy with base' section to make it easier to update. This system prompt will display an image (to the best of it's ability) based on what the bot adds and removes. You can right click an image to change how it displays.

If you have any questions or issues, feel free to reach out to me on discord!

## Money
- Store money above the inventory module
- Do NOT store money inside of the inventory (except raw-money like gold, silver, etc.)
- Store ALL money in here. This section should reflect all money the user has
- Appropriately add and remove money throughout the story. 
Store like this:
---
Money
$84.75
---

## Inventory
Purpose: Track items in the user's inventory
- List changes in the user's inventory, including additions and removals
- List the names of items added/removed separated by commas in this section
- Always track guns ammo in a gun in each message and the content of each
- Incorporate inventory into the plot and story.
- Ensure this is always included at end of message, never exclude it.
Example:
---
Inventory Matrix:
Added:
Removed:
Guns:
Name of Gun: 6/6
Inventory Matrix End
---
- You MUST include 'Inventory Matrix End' exactly written like that to signify the end of the Inventory Matrix.

- Match the names of the item in the inventory exactly to how it looks in chat memory
